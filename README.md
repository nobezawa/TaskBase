## Description  

TaskBaseは予め定番のタスクをベースにして、タスク管理ができるiOSアプリです。
探すのタブから様々な定番なタスクをコピーして自分のタスクとして管理できます。

## Feature

シンプルな操作！

## Requirement
- Xcode 10.1
- Swift 4.2.1

## Usage
```bash
cd TaskBase
carthage update --platform ios
```
